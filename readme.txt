Allows visitors to populate business letter template providing their custom attributes via form.

== Description ==

Letter Template allows you to setup a template for business letter with a placeholders for your visitors to fill up with their personal data using form fields and later download the ready to mail letter as a printable *.doc file.
Also it allows you to build a list of recipient Businesses with their name and address for the visitor to choose from.

[Demo](http://www.opzegbrief-autoverzekering.nl/)

**Main Functions and Features**

* Manage list of recipient Businesses;
* Build as many form fields for the visitor to fill up.
* Set up Business recipient template;
* Set up Business Letter Template with placeholder fields for the visitor to fill up.

Field types currently supported:

* Short Text - a single line input;
* Multi-line Text Area - self explanatory;
* Predefined Text - a checkbox to show/hide specific part of the letter content;
* Date - a date calendar input;
* Business Select - a dropdown of recipient Businesses;

== Installation ==

* Extract "wn-letter-template.zip" archive.
* Upload the "wn-letter-template" folder via FTP to the "/wp-content/plugins/" directory of your WordPress installation.
* In your WordPress administration, go to the Plugins page.
* Activate the "Letter Template" plugin through the 'Plugins' -- a new "LT-Businesses" menu will appear in your admin panel menus and a new "Letter Template" submenu will appear under your admin Settings menu.
